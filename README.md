# Hi, I'm David 👋

Security engineer building open source tools for SOC analysts and incident responders.

My focus: accessible, client-side tooling that covers the full analyst workflow — no accounts, no telemetry, no vendor lock-in.

---

## 🛡️ SecOps AI Suite

Six connected tools. One pipeline.

```
🧪 Extract  →  🔭 Pivot  →  🔬 Enrich  →  🚨 Explain  →  ✍️ Detect  →  📖 Respond
```

**[→ dgiry.github.io](https://dgiry.github.io)**

| Tool | What it does |
|------|-------------|
| [🧪 Defang IOC](https://dgiry.github.io/defang-ioc) | Extract and defang IOCs from any text |
| [🔭 IOC Pivot Hub](https://dgiry.github.io/ioc-pivot) | One indicator → 30+ threat intel platforms |
| [🔬 CVE Enricher](https://dgiry.github.io/cve-enricher) | CVSS · KEV · threat actors · patch priority |
| [🚨 Alert Explainer](https://dgiry.github.io/alert-explainer) | SIEM alert → plain English + MITRE mapping |
| [✍️ SIGMA Generator](https://dgiry.github.io/sigma-generator) | Attack description → SIGMA rule on 7 platforms |
| [📖 Playbook Builder](https://dgiry.github.io/playbook-builder) | MITRE technique → structured 6-phase IR draft |

All tools are free, MIT licensed, and run entirely in the browser.  
Supports **Trend Vision One XDR**, Microsoft Defender XDR, Splunk, Elastic, and more.
